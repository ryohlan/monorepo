# react-native-template

Template project for react-native.

This  project includes following configurations or libs:

- typescript
- eslint
- prettier
- react-navigation
- react-native-conifg
- redux
- react-native-debugger
- storybook

## Usage

### Start react-native application

```
yarn start
react-native run-[ios|android]
```

### Start storybook

```
yarn start:test
react-native run-[ios|android]
```
